# Neighbour Signals

A responsive, single-player community story game for research with older adults in Moray.

## Participant Flow

1. Enter an anonymous participant code and consent to research use.
2. Choose to answer as yourself or through a story character.
3. Select a familiar daily-life moment.
4. Place the moment on the Moray community map.
5. Reflect on feeling, frequency, importance, support, technology, privacy, and an optional note.
6. Review the personal connection map, add another story, or finish.

## Data Collected

Each completed story creates one anonymous record containing:

- participant code, optional age range, and broad Moray area
- role and scenario
- time context
- community node and node type
- feeling and frequency
- importance label and 1–4 score
- preferred support
- attitude to technology
- privacy boundary
- optional free-text note
- timestamp and game version

The research view summarises participants, story count, high-importance signals, technology openness, community nodes, support needs, and feelings. Data can be exported as CSV or JSON.

## Open And Deploy

The site has no build step and no external dependencies. Open `index.html` directly for a desktop test. For phone testing and offline installation, upload this entire folder to any static host such as Netlify, Vercel, GitHub Pages, or a university web server.
